# Queue Folder

## Purpose

This folder contains the project queue tracking system for managing work items across different categories (Research, Enhancement, Mitigation, Fix, Deliverable).

---

## Quick Reference

### Queue Numbering

- **RES-###**: Research items
- **ENH-###**: Enhancement items
- **MIT-###**: Mitigation items
- **FIX-###**: Fix items
- **DEL-###**: Deliverable items

### Status Values

- **Pending**: Not yet started
- **In Progress**: Active work ongoing
- **Completed**: Finished and verified
- **Blocked**: Cannot proceed due to blockers
- **Cancelled**: No longer needed

### Priority Levels

- **High**: Critical, blocking, or time-sensitive
- **Medium**: Important but not blocking
- **Low**: Nice to have, can be deferred

---

## Integration with Sessions

- **Session Start**: Review queue for high-priority pending items
- **During Session**: Update queue items as work progresses
- **Session End**: Document queue items addressed in session notes
