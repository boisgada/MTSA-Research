## Usage Guidelines

### For AI Sessions
1. **Start**: Review `ai-collaboration/project-context.md` and `ai-collaboration/rules-of-engagement.md`
2. **During**: Reference existing documents in appropriate folders
3. **End**: Document session using `ai-collaboration/session-notes-template.md`

### For Document Management
1. **Research**: Add research documents to `research/` subfolders
2. **Implementation**: Add guides to `implementation/guides/`, strategies to `implementation/strategies/`
3. **Assets**: Add supporting files to `assets/` subfolders
4. **Update READMEs**: Update relevant README files when adding documents

### For Navigation
1. **Quick Start**: Use `QUICK_REFERENCE.md` for common tasks
2. **Main Overview**: Start with main `README.md`
3. **Research**: Browse `research/` folder for analysis documents
4. **Implementation**: Browse `implementation/` folder for guides and strategies
5. **AI Collaboration**: Review `ai-collaboration/` folder for session guidelines
6. **Verified References**: Check `VERIFIED_REFERENCES.md` for source citations
