# Recommended GitHub Repositories for Reference

## Purpose

This document lists publicly available GitHub repositories that demonstrate useful patterns, techniques, and structures for AI-assisted projects.

---

## Suggested Categories

### AI / ML Project Templates

- Repositories with strong folder structure, CI/CD, testing, and containerization patterns
- Projects with good prompt, evaluation, or agent organization

### Data Science Project Templates

- Repositories that organize notebooks, data, reports, and models cleanly

### AI Best Practices and Guides

- Repositories focused on coding standards, testing, review practices, and agent guidance

### Awesome Lists and Collections

- Curated repositories that help discover tools, frameworks, and reusable patterns

---

## How to Use

1. Review repository structure for ideas worth adapting.
2. Extract patterns, not project-specific assumptions.
3. Document what was useful before adopting it here.
4. Prefer reusable structure and workflow patterns over direct copying.
