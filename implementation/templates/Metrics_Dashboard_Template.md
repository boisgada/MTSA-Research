# Compliance Metrics Dashboard Template

## Purpose

This template provides a structure for creating compliance metrics dashboards to monitor MTSA compliance status and progress.

---

## Dashboard Structure

### 1. Overall Compliance Status
- Overall compliance score (%)
- Compliance trend (improving/stable/declining)
- Critical gaps count
- High-priority gaps count

### 2. Requirement-by-Requirement Status

**For Each of 8 MTSA Requirements:**
- Requirement name
- Compliance status (%)
- Status indicator (Green/Yellow/Red)
- Key metrics
- Trend

### 3. Key Metrics Summary
- Training completion rate
- Assessment completion status
- Plan submission status
- Incident reporting compliance
- Drill/exercise completion

### 4. Risk Indicators
- Critical risks
- High-priority risks
- Risk trends
- Risk mitigation progress

### 5. Timeline Tracking
- Deadline tracking
- Milestone status
- Implementation progress
- Critical path items

### 6. Action Items
- Open action items
- Overdue items
- Upcoming deadlines
- Priority actions

---

## Template Usage

**Dashboard Creation:**
- Use spreadsheet or dashboard tool
- Update regularly (weekly/monthly)
- Include visual indicators
- Make accessible to stakeholders

**Monitoring:**
- Review regularly
- Identify trends
- Address issues promptly
- Report to management
