# Cybersecurity Assessment Report Template

## Purpose

This template provides a structure for documenting annual Cybersecurity Assessments required under MTSA. Customize for your specific facility and assessment methodology.

> Terminology: see `GLOSSARY.md`  
> Evidence: see `EVIDENCE_PACK_CHECKLIST.md` and `implementation/strategies/Cross_Framework_Evidence_Matrix.md`

---

## Assessment Report Structure

### 1. Executive Summary

- Assessment overview
- Key findings
- Risk summary
- Recommendations

### 2. Assessment Information

- Assessment date
- Assessment scope
- Assessment methodology
- Assessment team
- Facility information

### 3. Asset Inventory

- Critical assets identified
- Asset descriptions
- System interdependencies
- Asset classification

### 4. Vulnerability Assessment

- Vulnerabilities identified
- Vulnerability severity
- Affected systems
- Exploitability

### 5. Threat Assessment

- Threats identified
- Threat likelihood
- Threat impact
- Threat actors

### 6. Risk Assessment

- Risks identified
- Risk analysis
- Risk prioritization
- Risk treatment

### 7. Control Evaluation

- Existing controls
- Control effectiveness
- Control gaps
- Control recommendations

### 8. Findings and Recommendations

- Key findings
- Recommendations
- Prioritization
- Corrective actions

### 9. Corrective Action Plan

- Corrective actions
- Responsibilities
- Timelines
- Resource requirements

### 10. Appendices

- Detailed findings
- Supporting documentation
- Methodology details
- Additional information

---

## Template Usage

**Customization:**

- Adapt for your assessment methodology
- Include facility-specific findings
- Customize risk assessment approach
- Add supporting documentation

**Completion:**

- Complete all sections
- Provide sufficient detail
- Document all findings
- Include corrective actions
