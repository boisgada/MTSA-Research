# MTSA Compliance Self-Assessment Template

## Purpose

This template provides a structure for conducting self-assessments to evaluate MTSA compliance readiness. Use for pre-inspection preparation and continuous improvement.

---

## Self-Assessment Structure

### 1. Assessment Information
- Assessment date
- Assessment scope
- Assessor(s)
- Assessment methodology

### 2. Requirement-by-Requirement Review

**For Each of 8 MTSA Requirements:**

#### Requirement [Number]: [Name]
- Current implementation status
- Evidence of implementation
- Gaps identified
- Risk assessment
- Corrective actions needed

### 3. Overall Compliance Status
- Compliance score/rating
- Critical gaps
- High-priority gaps
- Medium-priority gaps
- Low-priority gaps

### 4. Gap Summary
- Total gaps identified
- Gaps by priority
- Gaps by requirement area
- Critical gaps requiring immediate attention

### 5. Corrective Action Plan
- Prioritized corrective actions
- Responsibilities
- Timelines
- Resource requirements

### 6. Readiness Assessment
- Inspection readiness
- Documentation readiness
- Personnel readiness
- Implementation readiness

### 7. Recommendations
- Immediate actions
- Short-term improvements
- Long-term enhancements
- Best practices

### 8. Appendices
- Detailed findings
- Evidence documentation
- Supporting materials

---

## Template Usage

**Assessment:**
- Conduct regularly (pre-inspection, annually)
- Be thorough and honest
- Document all findings
- Prioritize gaps

**Follow-Up:**
- Develop corrective actions
- Implement improvements
- Track progress
- Reassess regularly
