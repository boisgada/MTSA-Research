# Drill and Exercise Documentation Template

## Purpose

This template provides structure for documenting cybersecurity drills and exercises required under MTSA. Use for both drills (2 per year) and exercises (1 per year).

> Terminology: see `GLOSSARY.md`  
> Evidence: see `EVIDENCE_PACK_CHECKLIST.md` and `implementation/strategies/Cross_Framework_Evidence_Matrix.md`  
> Scenarios/visuals: see `implementation/strategies/MTSA_Examples_and_Diagrams.md`

---

## Drill/Exercise Documentation Structure

### 1. Drill/Exercise Information

- Type (drill or exercise)
- Date and time
- Duration
- Location
- Participants

### 2. Objectives

- Primary objectives
- Secondary objectives
- Success criteria
- Learning goals

### 3. Scenario

- Scenario description
- Threat/incident type
- Simulated conditions
- Realism factors

### 4. Execution

- Activities conducted
- Response actions
- Team coordination
- Timeline of events

### 5. Results

- Objectives achieved
- Performance assessment
- Strengths identified
- Areas for improvement

### 6. After-Action Review

- What went well
- What could be improved
- Lessons learned
- Recommendations

### 7. Corrective Actions

- Corrective actions identified
- Responsibilities
- Timelines
- Follow-up

### 8. Appendices

- Supporting documentation
- Participant lists
- Communications logs
- Additional information

---

## Template Usage

**For Drills:**

- Focus on specific procedures
- Test response capabilities
- Validate procedures
- Identify improvements

**For Exercises:**

- Comprehensive scenarios
- Full response testing
- Coordination testing
- End-to-end validation
