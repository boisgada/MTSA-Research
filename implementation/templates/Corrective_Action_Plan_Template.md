# Corrective Action Plan Template

## Purpose

This template provides a structure for developing corrective action plans to address identified gaps, vulnerabilities, or deficiencies. Use for tracking remediation efforts.

---

## Corrective Action Plan Structure

### 1. Finding Information
- Finding ID
- Finding description
- Source (Assessment/Inspection/Incident)
- Date identified
- Severity

### 2. Root Cause Analysis
- Root cause identified
- Contributing factors
- Analysis methodology
- Findings

### 3. Corrective Actions
- Action description
- Action objectives
- Specific tasks
- Approach/methodology

### 4. Implementation
- Responsibilities
- Resources required
- Timeline
- Milestones
- Dependencies

### 5. Verification
- Verification method
- Success criteria
- Verification date
- Verification results

### 6. Status Tracking
- Status (Not Started/In Progress/Completed/On Hold)
- Progress updates
- Completion date
- Issues/risks

### 7. Appendices
- Supporting documentation
- Evidence
- Additional information

---

## Template Usage

**Planning:**
- Develop for each finding
- Define clear actions
- Set realistic timelines
- Assign responsibilities

**Tracking:**
- Update status regularly
- Monitor progress
- Verify completion
- Document results
