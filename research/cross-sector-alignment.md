## Purpose

Identify land-based sectors with similar cyber/physical regulatory pressures and list industry groups and references we can reuse to accelerate MTSA compliance with minimal extra effort.

## High-Value Sectors and Groups

- **Chemical (CFATS legacy / hazmat)**
  - Groups: American Chemistry Council (ACC); Chemical Sector Coordinating Council (CSCC).
  - Use: Responsible Care¬Æ Security Code playbooks; CFATS-style facility security plans.
  - Sources: GAO CFATS/chemical security review (GAO-05-327) ‚Äî [govinfo GAO-05-327](https://www.govinfo.gov/content/pkg/GAOREPORTS-GAO-05-327/html/GAOREPORTS-GAO-05-327.htm)

- **Water/Wastewater**
  - Groups: American Water Works Association (AWWA); Water Sector Coordinating Council (WSCC).
  - Use: Risk/resilience assessment templates, emergency response planning; good analogs for facility drills and exercises.
  - Sources: GAO water sector security challenges ‚Äî [govinfo GAO-05-327](https://www.govinfo.gov/content/pkg/GAOREPORTS-GAO-05-327/html/GAOREPORTS-GAO-05-327.htm)

- **Energy (Electric + Oil/Gas)**
  - Groups: North American Electric Reliability Corporation (NERC); American Petroleum Institute (API).
  - Use: NERC CIP evidence practices for OT controls, incident reporting models; API cyber task guidance for OT and pipeline operations.
  - Sources: Critical infrastructure cyber risk (OT/third-party) ‚Äî [McKinsey critical infrastructure cyber risk](https://www.mckinsey.com/capabilities/risk-and-resilience/our-insights/critical-infrastructure-companies-and-the-global-cybersecurity-threat)

- **Refining / Petrochemical**
  - Groups: American Fuel & Petrochemical Manufacturers (AFPM).
  - Use: AFPM cybersecurity program (CISA CISP/ICSJWG participation, LOGIIC), annual Security Conference content, and APS (Advancing Process Safety) lessons learned for control systems and incident prevention.
  - Sources: AFPM cybersecurity overview ‚Äî [AFPM cybersecurity](https://www.afpm.org/issues/security/cybersecurity)
    - AFPM Security Conference (program topics include cybersecurity) ‚Äî [AFPM Security Conference](https://afpm.org/events/SC25)
    - AFPM sustainability report (APS outcomes) ‚Äî [AFPM 2022 sustainability report](https://afpm.org/system/files/attachments/WEB_FINAL_AFPM_2022_Sustainability_Report_220228.pdf)

- **Rail/Public Transit**
  - Groups: American Public Transportation Association (APTA).
  - Use: TSA surface transportation security directive playbooks (plans, drills, incident reporting) as parallels to MTSA plan/assessment cadence.
  - Sources: TSA rail security directive coverage (public guidance; align when available).

- **Security Technology / Converged Controls**
  - Groups: Security Industry Association (SIA).
  - Use: Access control/IDS standards and training; vendor ecosystem guidance for perimeters and regulated inspections.
  - Sources: SIA overview ‚Äî [securityindustry.org](https://www.securityindustry.org/)

- **Manufacturing / Industrial OT**
  - Groups: National Association of Manufacturers (NAM).
  - Use: OT segmentation patterns, OEM/third-party risk practices; useful for port OT stacks.
  - Sources: OT attack-surface challenges ‚Äî [tech-channels 2025 challenges](https://www.tech-channels.com/top-10/top-5-industries-facing-persistent-cybersecurity-challenges-in-2025)

- **Aviation**
  - Groups: Airports Council International (ACI World), ACI-NA.
  - Use: Airport cyber resilience guidance, access control and perimeter practices, and incident coordination models similar to port operations.
  - Sources: ACI aviation cybersecurity guidance (public summaries; pull official docs when available).

- **Aviation Security Info-Sharing**
  - Groups: Aviation ISAC (A-ISAC).
  - Use: Threat intel, incident sharing, and playbooks for cyber/physical convergence.
  - Sources: A-ISAC member resources (public summaries; requires membership for details).

- **Surface Transportation / Multi-Modal**
  - Groups: InfraGard (FBI/private sector partnership).
  - Use: Info sharing on threats to critical infrastructure, including transportation nodes.
  - Sources: [InfraGard overview](https://www.infragard.org/) (membership required for full access).

- **Industrial Control Standards**
  - Groups: ISA/IEC 62443 community (International Society of Automation).
  - Use: Control-system security standards and zones/conduits models directly applicable to OT networks in port facilities.
  - Sources: ISA/IEC 62443 (purchase/official channels).

- **Industrial Internet / CPS**
  - Groups: Industry IoT Consortium (formerly IIC).
  - Use: Cyber-physical systems security patterns and reference architectures.
  - Sources: IIC/IIOC public whitepapers.

- **Process Control Collaboration**
  - Groups: ICSJWG (CISA/DHS community); Process Control Systems Forum.
  - Use: Cross-sector control-system security practices, threat briefings, and roadmap materials.
  - Sources: ICSJWG public artifacts and roadmaps.

- **MTSA-Adjacent Intermodal Logistics**
  - Groups: Intermodal freight associations; port trucking/drayage councils; rail ramp operators (often coordinate via APTA freight committees or local port security committees).
  - Use: Cargo flow continuity, truck/rail access control, incident coordination, and credentialing practices that interface with MTSA-regulated facilities (gates, TWIC-aligned processes).
  - Sources: MTSA facility/vessel rules (33 CFR 101/105) drive joint drills and gate controls; pull local port security committee materials where available.

- **Marine Fuel/Bunkering & Pipelines into Marine Terminals**
  - Groups: API (pipeline and terminal committees); AFPM members supplying marine fuels.
  - Use: Pipeline terminal cyber/physical controls, loading rack practices, and product transfer procedures that must align with MTSA Facility Security Plans.
  - Sources: API terminal/pipeline guidance; MTSA 33 CFR 105 coordination requirements for transfer areas.

- **Passenger Terminals (Ferry/Cruise)**
  - Groups: Passenger Vessel Association (PVA); APTA ferries; ACI for passenger terminal security analogs.
  - Use: Crowd/embarkation security, screening, incident response, and public communications; parallels to MTSA-regulated passenger facilities.
  - Sources: MTSA Subchapter H (Parts 101, 104, 105) applicability to passenger terminals; APTA passenger security practices for drills/comms.
